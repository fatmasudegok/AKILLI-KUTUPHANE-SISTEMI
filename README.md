# Akıllı Kütüphane Sistemi
Bu proje, C# programlama dili ve .NET platformu kullanılarak Visual Studio 2022 ortamında geliştirilen Akıllı Kütüphane Sistemi isimli bir otomasyon yazılımıdır. Temel amacı kütüphane envanterini dijitalleştirmek olan bu sistem; kitap kayıtlarının tutulması, üye yönetiminin sağlanması ve ödünç alma süreçlerinin sistematik bir şekilde takip edilmesi için tasarlanmıştır. Modern yazılım geliştirme standartlarına uygun bir yapı sunan proje, hem yönetimsel işlemleri hızlandırmayı hem de kullanıcı deneyimini iyileştirmeyi hedefleyen bir çözüm sunmaktadır.
